🔑 API Key 등록 가이드

잇-Story는 사용자가 직접 발급한 AI API 키로 분석을 수행합니다. 앱이 개인 데이터를 수집하지 않는 이유입니다.

💡 왜 API 키가 필요한가요?

잇-Story는 AI 분석 서비스를 위해 별도의 서버를 운영하지 않습니다. 대신, 사용자가 직접 Google, OpenAI, Anthropic에서 발급받은 API 키를 기기 내에 암호화 저장하여 AI 서비스를 직접 호출합니다. 이를 통해 개인 식단 데이터가 외부로 전송되지 않도록 설계되어 있습니다.

📱 Google Gemini API 키 등록 9단계

1

Google AI Studio 접속

aistudio.google.com에 접속하여 Google 계정으로 로그인합니다. 좌측 상단의 ☰ 메뉴 버튼을 탭합니다.

💡 Google 계정이 필요합니다. 없으시면 먼저 가입하세요.
2

Get API key 메뉴 선택

사이드 메뉴 하단에 있는 "Get API key" 메뉴를 탭합니다.

🔑 API 키 관리 페이지로 이동합니다.
3

API 키 만들기 버튼 클릭

API 키 목록 화면에서 우측 상단의 "API 키 만들기" 버튼을 탭합니다.

📋 이미 키가 있다면 기존 키를 사용하셔도 좋습니다.
4

키 이름 입력

"새 키 만들기" 팝업에서 이름을 입력(예: eatstory)한 후 "키 만들기"를 탭합니다.

✏️ 키 이름은 나중에 구분하기 위한 것으로 자유롭게 입력하세요.
5

API 키 생성 확인

키가 성공적으로 생성되었습니다! 목록에서 방금 만든 키를 탭하여 세부정보를 확인합니다.

6

API 키 복사

API 키 값 옆의 📋 복사 버튼을 탭하여 키를 클립보드에 복사합니다.

⚠️ API 키는 비밀번호처럼 소중히 다뤄주세요! 타인에게 공유하지 마세요.
7

잇-Story 앱 실행

잇-Story 앱을 실행합니다. 메인 화면이 나타나면 하단 네비게이션에서 "설정" 탭으로 이동합니다.

8

기능 설정 탭 선택

설정 페이지 상단의 "기능 설정" 탭을 선택합니다.

⚙️ 기능 설정 탭에서 AI 서비스를 관리할 수 있습니다.
9

API 키 입력 및 저장

"지능형 AI 관리 서비스" 섹션에서 Google Gemini 항목의 API 키 입력란에 복사한 키를 붙여넣고 "저장" 버튼을 탭합니다.

🎉 "인증 완료"가 표시되면 설정 완료! 이제 AI 식단 분석을 바로 사용할 수 있습니다.

⚠️ 무료 API 사용 제한 안내

Gemini 무료(Free Tier) 버전은 API 키 단위가 아닌 프로젝트 단위로 제한이 적용됩니다.

  • 모델별 하루 약 20~50회 요청 가능 (구글 정책에 따라 변동)
  • 2025년 12월 이후 무료 한도가 대폭 축소된 상태
  • 한도 초과 시 분석 지연 또는 오류 발생 가능

✅ Smart Routing으로 한도 극복

잇-Story는 3가지 Gemini 모델(1.5 / 2.5 / 2.5 Lite)을 자동 전환하여 하루 실질 사용 가능 횟수를 최대 약 60~150회까지 확장합니다. 더 많은 분석이 필요하다면 유료 플랜으로 전환하시거나, OpenAI·Claude API 키를 추가로 등록하세요.

🔑 API Key Registration Guide

Eat-Story uses AI API keys that you generate yourself — that's how your data stays private.

💡 Why do I need an API key?

Eat-Story does not operate its own servers for AI analysis. Instead, your API keys (issued directly by Google, OpenAI, or Anthropic) are encrypted and stored on your device, and AI services are called directly from your device. This ensures your personal meal data never leaves your device.

📱 Google Gemini API Key — 9 Steps

1

Go to Google AI Studio

Visit aistudio.google.com and sign in with your Google account. Tap the ☰ menu button at the top-left.

💡 A Google account is required.
2

Select "Get API key"

Tap "Get API key" from the sidebar menu.

🔑 This takes you to the API key management page.
3

Create API Key

Tap "Create API key" in the top-right area of the key list.

📋 If you already have a key, you can reuse it.
4

Enter a key name

In the popup, enter a name (e.g., "eatstory") and tap "Create key".

✏️ The name is just for your reference — use anything.
5

Confirm key created

Your key has been created! Tap the new key in the list to see its details.

6

Copy the API key

Tap the 📋 copy button next to the API key value.

⚠️ Treat your API key like a password — never share it!
7

Open Eat-Story app

Launch the Eat-Story app. From the main screen, tap "Settings" in the bottom navigation.

8

Go to Feature Settings tab

Select the "Feature Settings" tab at the top of the Settings page.

⚙️ AI services are managed in this tab.
9

Paste and save the API key

In the "AI Management" section, paste your key into the Google Gemini field and tap "Save".

🎉 When "Verified" appears, you're all set! AI meal analysis is ready to use.

⚠️ Free API Usage Limits

Gemini Free Tier limits apply per project, not per API key.

  • Approximately 20–50 requests per day per model (varies by Google policy)
  • Free tier quotas were significantly reduced after December 2025
  • Exceeding the limit may cause analysis delays or errors

✅ Overcome Limits with Smart Routing

Eat-Story automatically rotates between 3 Gemini models (1.5 / 2.5 / 2.5 Lite), effectively extending your daily usable requests to approximately 60–150 per day. For heavier usage, switch to a paid plan or add keys for OpenAI or Claude.
