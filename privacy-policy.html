개요

'잇-Story' (이하 "앱")는 개인 개발자 홍원기가 제공합니다. 본 방침은 앱 이용 시 수집 및 처리하는 정보에 대해 안내합니다. 앱의 모든 데이터는 사용자의 기기 내에만 저장되며, 별도의 서버에 전송하거나 보관하지 않습니다.

외부 AI 서비스 이용 (제3자 데이터 전송)

앱은 식사 분석 및 스토리 생성을 위해 아래 AI 서비스에 데이터를 전송할 수 있습니다. 서비스는 사용자가 직접 선택 및 API 키를 입력하며, 전송 여부는 사용자가 통제합니다.

  • Google Gemini API — 식사 사진 분석, 텍스트 생성 (Google 개인정보처리방침)
  • Anthropic Claude API (선택) — 텍스트 생성 (Anthropic 개인정보처리방침)
  • OpenAI API (선택) — 텍스트 생성 (OpenAI 개인정보처리방침)
⚠️ AI 서비스에 전송된 데이터(사진, 텍스트)는 해당 서비스의 개인정보 정책에 따라 처리됩니다. 민감한 개인정보의 포함 여부에 주의해 주세요.

기기 권한

  • 카메라: 음식 사진 촬영 (선택 사항)
  • 사진첩: 음식 사진 선택 및 저장 (선택 사항)

모든 권한은 사용자 동의 시에만 요청되며, 거부해도 앱의 기본 기능 이용이 가능합니다.

데이터 보유 및 삭제

모든 데이터는 사용자가 직접 삭제하거나 앱을 삭제할 때까지 기기에 보관됩니다. 앱 삭제 시 모든 데이터가 함께 삭제됩니다. 앱 내의 설정 > 데이터 관리 메뉴에서 언제든 데이터를 초기화할 수 있습니다.

미성년자 보호

Overview

'Eat-Story' (the "App") is provided by individual developer Wongi Hong. This policy describes how the App handles your information. All data is stored locally on your device only and is never transmitted to or stored on external servers (except as noted below for AI services).

Information We Collect & Why

Data Purpose Storage
Meal records (food name, calories, etc.) Meal tracking & AI analysis On-device (encrypted)
Food photos AI nutrition analysis, records On-device (encrypted)
Weight records Weight tracking & statistics On-device
Water intake records Hydration tracking On-device
Alcohol records Lifestyle habit tracking On-device
App settings & preferences Personalized experience On-device
AI API keys (optional, user-provided) Authentication with AI services On-device (encrypted)
Body information (age, gender, height, weight, target weight) Calorie goal calculation, personalized AI analysis On-device (encrypted)
Health goals & activity level Personalized AI coaching, daily nutrient targets On-device (encrypted)
Fasting records (target hours, achievement) Intermittent fasting tracking feature On-device (encrypted)

Third-Party AI Services

The App may transmit data to the following AI services for meal analysis and story generation. This requires a user-provided API key and is entirely under the user's control.

  • Google Gemini API — Photo & text analysis (Google Privacy Policy)
  • Anthropic Claude API (optional) — Text generation (Anthropic Privacy Policy)
  • OpenAI API (optional) — Text generation (OpenAI Privacy Policy)
⚠️ Data (photos, text) sent to AI services is subject to those services' own privacy policies. Please be mindful of sensitive personal information included in your meal photos or notes.

Device Permissions

  • Camera: To take food photos (optional)
  • Photo Library: To select and save food photos (optional)

All permissions are optional and requested only when needed. You can use core app features without granting them.

Data Retention & Deletion

All data is stored on your device until you delete it or uninstall the App. Uninstalling the App removes all associated data. You can also clear all data anytime from Settings > Data Management within the App.

Children's Privacy

This App is not directed to children under 14 years of age, and we do not knowingly collect personal information from children.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be communicated via app updates or this page. Significant changes will be announced via in-app notifications.

外部AIサービスの利用（第三者へのデータ転送）

本アプリは、食事分析およびストーリー生成のために、以下のAIサービスにデータを転送する場合があります。サービスはユーザーが直接選択し、APIキーを入力して利用するため、転送の有無はユーザーが制御します。

  • Google Gemini API — 写真・テキスト分析 (Google プライバシーポリシー)
  • Anthropic Claude API (任意) — テキスト生成 (Anthropic プライバシーポリシー)
  • OpenAI API (任意) — テキスト生成 (OpenAI プライバシーポリシー)
⚠️ AIサービスに送信されたデータ（写真、テキスト）は、各サービスのプライバシーポリシーに従って処理されます。機密性の高い個人情報の含まれ方に注意してください。

デバイス権限

  • カメラ：食事写真の撮影（オプション）
  • 写真ライブラリ：食事写真の選択および保存（オプション）

すべての権限はユーザーの同意がある場合にのみ要求され、拒否してもアプリの基本機能の利用は可能です。

データの保持および削除

すべてのデータは、ユーザーが直接削除するか、アプリをアンインストールするまでデバイスに保持されます。アプリ削除時、すべてのデータが共に削除されます。アプリ内の 設定 > データ管理 メニューからいつでもデータを初期化できます。

未成年者の保護

本アプリは14歳未満の子供を対象としておらず、意図的に子供の個人情報を収集することはありません。
