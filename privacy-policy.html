개요

'잇-Story' (이하 "앱")는 개인 개발자 홍원기가 제공합니다. 본 방침은 앱 이용 시 수집 및 처리하는 정보에 대해 안내합니다. 앱의 모든 데이터는 사용자의 기기 내에만 저장되며, 별도의 서버에 전송하거나 보관하지 않습니다.

항목 이용 목적 저장 위치

외부 AI 서비스 이용 (제3자 데이터 전송)

앱은 식사 분석, 건강 레포트 생성, 식단 기반 스토리 창작을 위해 아래 AI 서비스에 데이터를 전송할 수 있습니다. 서비스는 사용자가 직접 선택 및 API 키를 입력하며, 최초 분석 시도 시 앱 내에서 명시적인 동의를 받은 후에만 데이터가 전송됩니다.

📡 데이터 수신 대상

  • Google Gemini API — 식사 사진 분석, 텍스트 생성 (Google 개인정보처리방침)
  • Anthropic Claude API (선택) — 텍스트 생성 (Anthropic 개인정보처리방침)
  • OpenAI API (선택) — 텍스트 생성 (OpenAI 개인정보처리방침)

📋 전송되는 데이터 항목

  • 식사 데이터: 식사 사진, 식사 설명(텍스트), 날짜/시간, 식사량, 조리 방식, 추가 메모
  • 프로필 정보: 닉네임, 직업, 나이, 성별, 키, 현재 체중, 목표 체중
  • 건강 정보: 활동량, 건강 관심사, 건강검진 특이사항, 건강 목표
  • 활동 데이터: 체중 변화 기록, 수분 섭취 기록, 금주 기록 (통계 분석용)

🎯 데이터 전송 목적

개인 맞춤형 영양 분석, 건강 레포트 생성, 식단 기반 스토리(소설/시/일기) 창작에 활용됩니다.

⚠️ 사전 동의 필수: AI 서비스에 데이터를 전송하기 전, 앱 내에서 전송되는 데이터 항목과 수신 대상을 명확히 안내하고 사용자의 명시적 동의를 받습니다. 동의하지 않으면 데이터가 전송되지 않습니다.
⚠️ AI 서비스에 전송된 데이터는 해당 서비스의 개인정보 정책에 따라 처리됩니다.
⚠️ 사용자는 설정 > 고급 데이터 관리에서 언제든지 동의를 철회할 수 있습니다.

기기 권한

  • 카메라: 음식 사진 촬영 (선택 사항)
  • 사진첩: 음식 사진 선택 및 저장 (선택 사항)

모든 권한은 사용자 동의 시에만 요청되며, 거부해도 앱의 기본 기능 이용이 가능합니다.

데이터 보유 및 삭제

Datenschutzrichtlinie

Letzte Aktualisierung: 20.03.2026

Einleitung

Eat-Story („App“) respektiert Ihre Privatsphäre und verarbeitet Ihre persönlichen Daten nach dem Prinzip der vollständigen On-Device-Speicherung. Die App sammelt standardmäßig keine persönlichen Informationen auf unseren Servern, und alle Ernährungsaufzeichnungen bleiben auf Ihrem Gerät.

Datensammlung und -nutzung

  • Ernährungsdaten und Fotos: Alle Ernährungsdaten (Fotos, Makronährstoffe) und Charakterentwicklungen (Nyam-i) werden nur lokal gespeichert.
  • Gesundheitstracker: Gewicht, Wasseraufnahme, Fastenzeiten etc. werden nur lokal auf dem Gerät gespeichert.
  • Die App speichert von selbst keine sensiblen Gesundheitsdaten, die für Werbezwecke missbraucht werden könnten.

KI Analyse & API-Key Verarbeitung (BYOK)

Diese App analysiert Mahlzeiten auf Basis von benutzerdefinierten API-Schlüsseln (Bring Your Own Key).

  • Google Gemini (Standard): Der von Ihnen eingegebene API-Schlüssel wird sicher und verschlüsselt auf Ihrem Gerät gespeichert. Fotos werden ausschließlich über die offizielle Google Gemini API zur Analyse gesendet.
  • OpenAI / Claude: Falls Sie GPT-4o oder Claude Models nutzen möchten, können Sie entsprechende Keys speichern. Auch diese verbleiben nur auf dem Gerät.
⚠️ An KI-Dienste gesendete Daten unterliegen deren jeweiligen Datenschutzrichtlinien.
⚠️ Sie können die Einwilligung jederzeit unter Einstellungen > Datenverwaltung widerrufen.

Geräteberechtigungen

  • Kamera: Aufnahme von Essensfotos (optional)
  • Fotobibliothek: Auswahl und Speicherung von Essensfotos (optional)

Alle Berechtigungen werden nur nach Zustimmung angefordert, und Sie können die App auch bei Ablehnung der Berechtigungen eingeschränkt nutzen.

Datenaufbewahrung und -löschung

Alle Daten werden auf Ihrem Gerät gespeichert, bis Sie die Anwendung deinstallieren oder die Daten manuell löschen. Die Daten können jederzeit über die Einstellungen > Datenverwaltung zurückgesetzt werden.

Schutz Minderjähriger

Diese Anwendung richtet sich nicht an Kinder unter 14 Jahren und wir sammeln nicht wissentlich persönliche Daten von Kindern.

Änderungen der Richtlinie

Bei Änderungen der Richtlinie informieren wir Sie über ein App-Update oder auf dieser Seite. Wesentliche Anpassungen können per In-App-Hinweis mitgeteilt werden.
